# A01: California Housing Boxplot

This project loads the California Housing dataset and generates a boxplot saved as a PNG figure.

## Data

The [California Housing dataset](https://scikit-learn.org/stable/datasets/real_world.html#california-housing-dataset) is fetched directly from scikit-learn (`fetch_california_housing`). No manual download is needed - the script handles it automatically.

## Setup

Install dependencies:

```bash
pip install -r requirements.txt
```

## How to Run

From the root of the repo:

```bash
python src/boxplot.py
```

## Expected Output

The script saves a boxplot to:

```
figs/boxplot.png
```
